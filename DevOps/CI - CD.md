# Pipeline
- Todos los pasos a seguir, por ejemplo
	- Se hace un commit/push a master
	- Se activa la compilación
	- Se activa el Empaquetado
	- Se hacen pruebas unitarias automatizadas
	- Se hacen pruebas de interfaz de usuario automatizadas
>[!info]
>Todos los pasos del pipeline deberian de estar automatizados y la intervención humana deberia de ser nula o poca.
